Tłumaczenia nazw i krótkie opisy ksiąg:¶

  • Book of Eccleasiasticus - Mdro Syracha - jedna z ksig deuterookanonicznych Starego Testamentu (czyli takich, ktre s w Biblii chrzecijaskiej, ale nie w hebrajskiej. Napisana ok. 190 r.p.n.e. w Jerozolimie
  • Book Of Ecclesiastes - Ksiga Koheleta - znajduje sie w obu Bibliach. Datowanie ksigi nie jest pewne, cho s znaki, aby okrela je na III w.p.n.e.
  • Book of Proverb - Ksiga Przysw - rwnie w obu Bibliach, jest prac zbiorow zoon z rnych tekstw przez nieznanego autora ok. V w.p.n.e.
  • Book of Wisdom - Ksiga Mdroci - napisana prawdopodobnie w Aleksandrii (Egipt) - datowana na ok. 50 r.p.n.e.
  • Buddhism - oczywicie nazwa religii dalekiego wschodu - buddyzmu, ktry najczciej jest wyznawany w kraja pwyspu indochiskiego, Malezji, Chinach i Monoglii.
  • Tao Te Ching - take: Lao Tsu - chiska ksiga najprawdopodobniej napisana w 6 wieku p.n.e. przez mdrca Laozi. Uwaana jest za podstawowe dzieo taoizmu - jedn z najpopularniejszych chiskich religii. Badacze twierdz take, e miaa ona wpyw take na ksztatowanie filozofii buddystw.
  • Upanishads - upaniszady - teksty (w liczbie ponad 200) religii hinduizmu nalece do wedyjskiego objawienia o treci religijno - filozoficznej. Jedne z pniejszych tekstw nalecych do Wed (witych ksig hinduizmu), napisane w VIII - III w.p.n.e. S do dzi wykorzystywane nawet w naszej kulturze wrd osb praktykujcych medytacj.
  • Yoga Sutra - jogasutry (hinduizm) - najstarszy tekst klasycznej Jogi (w znaczeniu systemu filozofii indyjskiej, uznajcego autorytet Wed), skadajcy si ze 195 sutr - zwizych sentencji. Wedug hinduskich tradycji, ich autorem by yjcy w II w.p.n.e. Patajli.

def lemmatization(data):
  '''
  Podajemy numpy array list a zwracana jest numpy array list unikatowych sw po lematyzacji
  '''
  lem = WordNetLemmatizer()
  A = []
  for i in data.tolist():
    a = lem.lemmatize(i)
    a = lem.lemmatize(a, 'a')
    a = lem.lemmatize(a, 'v')
    A.append(a)
  x = np.array(A)
  A = np.unique(x)
  return A

Opis metodologii:¶

Dla kadej z ksig moemy bez problemu zdoby wektor licznoci wystpie poszczeglnych sw. Dla kadej pary ksig mona zatem obliczy odlego euklidesow biorc za wsprzdne te wanie licznoci i porwna obliczone wartoci dla rnych par.

Takie dziaanie ma sens, gdy porwnujemy ksigi o podobnym rzdzie liczby wyrazw. Dla poszczeglnych tekstw, gdzie sw jest kilkadziesit, osigane wyniki byyby wysze.

def distance_plot(labels, df, ret_max = False, vmax = None):
    data = df/(df.sum(axis = 1)[:, None])
    A = [[0]*len(labels) for _ in range(len(labels))]
    maxdist = 0
    for i in range(len(labels)):
        for j in range(i+1, len(labels)):
            x = data.iloc[i,:]
            y = data.iloc[j,:]
            dist = math.sqrt(np.sum((x-y)**2))
            A[i][j] = dist
            A[j][i] = dist
            if dist > maxdist: maxdist = dist
    if ret_max: return maxdist
    sns.heatmap(A, annot = True, fmt = '.3f', vmax = vmax, xticklabels = labels, yticklabels=labels).set_title('"Oglegoci" midzy ksigami - im mniej, tym bardziej podobne')
    plt.show()

def sentiment(data):
  books_sentiment = []
  for row in data.values:
    i = 0
    pos, neu, neg = 0,0,0
    # zliczamy liczby sentymentw
    for x in row:
      if columns_sentiment[i] == "positive":
        pos = pos + x
      elif columns_sentiment[i] == "negative":
        neg = neg + x
      else:
        neu = neu + x
      i = i + 1
    # wynik dla wiersza
    if pos >= neg:
      if pos < neu or pos == neg:
        books_sentiment.append("neutral")
      else:
        books_sentiment.append("positive")
    else:
      if neg < neu:
        books_sentiment.append("neutral")
      else:
        books_sentiment.append("negative")
  return books_sentiment

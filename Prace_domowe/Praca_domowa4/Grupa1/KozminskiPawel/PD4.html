def test_pipeline(pipeline):
    param_distributions = {'svr__C': uniform(loc = 0.01, scale = 2),
                           'svr__gamma': expon(0.05),
                           'svr__degree': randint(low = 1, high = 3),
                           'svr__kernel': ['linear', 'poly', 'rbf', 'sigmoid'],
                           'svr__epsilon': uniform(0.005, scale = 0.12)}

    rscv = RandomizedSearchCV(pipeline, param_distributions=param_distributions,
                          scoring = 'neg_root_mean_squared_error',
                          cv = 4, random_state = 324, n_iter = 10,
                          verbose = 10**8)

    return rscv.fit(apart, train_target)

Uwagi co do strojenia hiperparametrów:¶

  • SVR nie posiada parametru cost. Zamiennie zastosowaem parametr C - dodatni parametr regularyzacji
  • Parametr gamma - sposb ustawienia wspczynnika dla jdra w przypadku gdy jest ono jednym z 'rbf' - Gaussian radial basis function, 'poly' - Polynomial, 'sigmoid'. Warto wspczynnika w przypadku gdy gamma = 'scale' jest zalena od rozmiaru danych wejciowych i wynosi 1/(n_features() * X.var()). Gdy 'gamma' = 'auto', jego warto to po prostu 1/n_features. Dla celw badawczych sprawdzimy jak spisz si wartoci tego parametru z rozkadu wykadniczego o parametrze 0.05).
  • Parametr degree - okrela stopie wielomianu dla jdra 'poly'. W przeciwnym razie - jest ignorowany. Co ciekawe, gdy degree byo liczb z zakresu 3-6, obliczenia trway nieporwnywalnie duej (adne nie doszo do skutku w rozsdnym czasie)
  • kernel - rodzaj jdra
  • epsilon - o ile dobrze rozumiem, jest to okrelenie pewnego marginesu, na ktrym nie jest stosowana adna kara za niepoprawn predykcj; moliwa pomyka bez kary
    W celu poprawnego stosowania kodowania zmiennej kateogrycznej w procesie poszukiwania optymalnych hiperparametrw, zastosujemy obiekt ColumnTransofrmer

def fifa_pipeline_test(ppl):
    param_distributions = {'svr__C': uniform(loc = 0.01, scale = 2),
                           'svr__gamma': expon(0.05),
                           'svr__degree': [1], # Tylko 1, gdy 'poly' natrafi na 2, czas dziaania na tak duym zbiorze rs dramatycznie
                           'svr__kernel': ['linear', 'poly', 'rbf', 'sigmoid'],
                           'svr__epsilon': uniform(0.02, scale = 0.3)}

    fifa_rscv = RandomizedSearchCV(ppl, param_distributions=param_distributions,
                              scoring = 'neg_root_mean_squared_error',
                              cv = 4, random_state = 794, n_iter = 10,
                              verbose = 10**8)

    return fifa_rscv.fit(data_train, target_train)
